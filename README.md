## ğŸš€ Getting Started

### Prerequisites
- Node.js (v18+)
- MongoDB (Local or Atlas)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/0paziz/SpendWise_MERN-Stack.git
   cd expense-tracker
   ```

2. **Frontend Setup:**
   ```bash
   cd frontend
   npm install
   cp .env.example .env # Update your variables
   npm run dev
   ```

3. **Backend Setup:**
   ```bash
   cd ../backend
   npm install
   cp .env.example .env # Update your variables
   node server.js
   ```
